[[build]]
Build
-----

[[requirements]]
Requirements
~~~~~~~~~~~~

You have to use node version 5.1.1 until we upgrade to current Gitbook
version.

[[install-all-dependencies]]
Install all dependencies
~~~~~~~~~~~~~~~~~~~~~~~~

....
npm install
....

[[build-static-page]]
Build static page
~~~~~~~~~~~~~~~~~

....
make build
open _book/index.html
....

[[publish-github-page]]
Publish github page
~~~~~~~~~~~~~~~~~~~

....
make publish
....
